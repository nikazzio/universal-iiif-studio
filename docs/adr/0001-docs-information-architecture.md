# ADR 0001: Documentation Information Architecture

## Status

Accepted

## Decision

Scriptoria documentation uses a content-first structure:

- `intro`
- `guides`
- `reference`
- `explanation`
- `project`
- `adr`

## Rationale

The previous structure was workable but too flat and too dependent on readers already knowing where to look.

This structure keeps the content model simple while still making the site easier to navigate and extend later.
